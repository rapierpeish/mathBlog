微分形式

微分形式（Differential Form）是流形上的反对称多重线性函数，是描述局部微分性质、高维积分及物理场（如力场、通量场）的核心数学工具。其本质是 “不依赖坐标系的微分语言”，能统一向量分析、微积分中的各类微分 / 积分运算（如梯度、散度、旋度、线面积分），也是连接微分几何、代数拓扑与物理模拟（如流体、布料）的关键桥梁 。

前置概念

  • 流形MMM：局部与欧氏空间 Rn\mathbb{R}^nRn 同胚的几何对象。
  • 切空间 TpMT_pMTp​M：流形上某点 ppp 处 “所有切向量的集合”，是 n- 维线性空间。
  • 余切空间 Tp∗MT^{*}_{p}MTp∗​M ：切空间 TpMT_pMTp​M 的[[线性空间的对偶空间#对偶空间|对偶空间]]，其元素余切向量是从切向量到实数的线性函数集合：

Tp∗M={f:TpM→R∣f is linear function}T^{*}_{p}M=\{f:T_pM\to\mathbb{R}| \text{f is linear function}\} Tp∗​M={f:Tp​M→R∣f is linear function}

多重线性函数

若函数 f:TpM×TpM×…TpM→Rf:T_{p}M\times T_{p}M\times\dots T_{p}M\to \mathbb{R}f:Tp​M×Tp​M×…Tp​M→R，满足以下两点，则 fff 就是切空间 TpMT_pMTp​M 上的n重线性函数：

  • 数乘性: f(v1,…kvi,…,vn)=kf(v1,…vi,…,vn),k∈Rf(v_{1},\dots kv_{i},\dots ,v_{n})=kf(v_{1},\dots v_{i},\dots ,v_{n}),k\in \mathbb{R}f(v1​,…kvi​,…,vn​)=kf(v1​,…vi​,…,vn​),k∈R

  • 加法性: f(v1,…,(vi+vj),…,vn)=f(v1,…,vi,…,vn)+f(v1,…,vj,…,vn)f(v_{1},\dots ,(v_{i}+v_{j}),\dots ,v_{n})=f(v_{1},\dots ,v_{i},\dots ,v_{n})+f(v_{1},\dots ,v_{j},\dots ,v_{n})f(v1​,…,(vi​+vj​),…,vn​)=f(v1​,…,vi​,…,vn​)+f(v1​,…,vj​,…,vn​)

反对称性

给定k重线性函数 fff ，如果fff 满足下列性质，则fff 是反对称的：

f(v1,…,vi,…,vj,…,vk)=−f(v1,…,vj,…,vi,…,vk)f(v_{1},\dots,v_{i},\dots,v_{j},\dots, v_{k})=-f(v_{1},\dots,v_{j},\dots,v_{i},\dots, v_{k}) f(v1​,…,vi​,…,vj​,…,vk​)=−f(v1​,…,vj​,…,vi​,…,vk​)

推论1: 若 fff 是反对称的，且两个分量相等(vi=vj,i≠jv_{i}=v_{j},i\neq jvi​=vj​,i=j)，则 f(v1,v2,…vk)=0f(v_{1},v_{2},\dots v_{k})=0f(v1​,v2​,…vk​)=0 推论2: 若 k>nk>nk>n，nnn 为线性空间的维数，则 VVV 上的反对称 kkk 重线性函数必为零函数。

推论1证明

如果 vi=vj,i≠jv_{i}=v_{j},i\neq jvi​=vj​,i=j，交换 viv_{i}vi​ 和 vjv_{j}vj​ 顺序得到：

f(v1,v2,…vi,…vj,…vk)=−f(v1,v2,…vj,…vi,…vk)f(v_{1},v_{2},\dots v_{i},\dots v_{j},\dots v_{k}) = -f(v_{1},v_{2},\dots v_{j},\dots v_{i},\dots v_{k}) f(v1​,v2​,…vi​,…vj​,…vk​)=−f(v1​,v2​,…vj​,…vi​,…vk​)

又因为 vi=vjv_{i}=v_{j}vi​=vj​ ，所以：

f(v1,v2,…vi,…vj,…vk)=f(v1,v2,…vj,…vi,…vk)f(v_{1},v_{2},\dots v_{i},\dots v_{j},\dots v_{k}) = f(v_{1},v_{2},\dots v_{j},\dots v_{i},\dots v_{k}) f(v1​,v2​,…vi​,…vj​,…vk​)=f(v1​,v2​,…vj​,…vi​,…vk​)

那么只能是：

f(v1,v2,…vi,…vj,…vk)=0f(v_{1},v_{2},\dots v_{i},\dots v_{j},\dots v_{k}) = 0 f(v1​,v2​,…vi​,…vj​,…vk​)=0

推论2证明 因为 k>nk>nk>n ，那么向量组 {v1,v2,…vk}\{v_{1},v_{2},\dots v_{k}\}{v1​,v2​,…vk​} 是线性无关了。所以存在 kkk 满足：

vk=αjvj,j∈{1,2,…k−1}v_{k} = \alpha^{j}v_{j}, j\in \{1,2,\dots k-1 \} vk​=αjvj​,j∈{1,2,…k−1}

αjvj\alpha^{j}v_{j}αjvj​ 是求和约定

根据双线性性质展开：

f(v1,v2,…vk)=f(v1,v2,…,αjvj)=αjf(v1,v2,…,vj),j∈{1,2,…k−1}\begin{align} f(v_{1},v_{2},\dots v_{k}) &= f(v_{1},v_{2},\dots, \alpha^{j}v_{j}) \\ &= \alpha^{j}f(v_{1},v_{2},\dots , v_{j}),j\in \{1,2,\dots k-1 \} \end{align} f(v1​,v2​,…vk​)​=f(v1​,v2​,…,αjvj​)=αjf(v1​,v2​,…,vj​),j∈{1,2,…k−1}​​

根据推论1，对任意 j∈{1,2,…k−1}j\in \{1,2,\dots k-1 \}j∈{1,2,…k−1} 都满足 f(v1,v2,…,vj)=0f(v_{1},v_{2},\dots , v_{j}) = 0f(v1​,v2​,…,vj​)=0 ，所以 f(v1,v2,…vk)f(v_{1},v_{2},\dots v_{k})f(v1​,v2​,…vk​) 是个零函数。

外幂空间

设 VVV 是 R\mathbb{R}R 上 nnn 维线性空间，VVV 的k 次外幂空间 Λk(V)\Lambda^k(V)Λk(V)，定义为VVV 上全体反对称 kkk 重线性函数的集合，即：

Λk(V)={f∈Lk(V,R)∣f是反对称k重线性函数}\Lambda^k(V)=\{f\in \mathcal{L}^{k}(V,\mathbb{R})\mid \text{f是反对称k重线性函数}\} Λk(V)={f∈Lk(V,R)∣f是反对称k重线性函数}

  • k=0k=0k=0 时：Λ0(V)=R\Lambda^0(V)=\mathbb{R}Λ0(V)=R 是0 重线性函数定义为常数函数，唯一反对称性是自身

ωp:TpM×TpM×⋯×TpM→R\omega_{p}:T_{p}M\times T_{p}M\times \dots \times T_{p}M\to \mathbb{R} ωp​:Tp​M×Tp​M×⋯×Tp​M→R

k-形式场是流形MMM 上每一点 ppp 的 k-形式集合，记为 Ωk(M)\Omega^{k}(M)Ωk(M)。
