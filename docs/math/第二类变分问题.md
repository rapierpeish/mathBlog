# 引言

## 1.1 变分问题分类

根据是否存在约束条件，变分问题可分为两类：

- **第一类变分问题**：无积分约束条件，仅需满足边界条件，核心是通过欧拉-拉格朗日方程求解泛函极值（如最速降线问题、短程线问题的基础形式）
- 2. **第二类变分问题**：又称等周问题或带约束变分问题，在满足一个或多个积分约束条件的前提下，求解泛函的极值。其本质是有约束的泛函优化问题，是第一类变分问题的拓展。

## 1.2 第二类变分问题的严格定义

设待求极值函数 $y(x) \in C^1[x_0, x_1]$，满足固定边界条件 $y(x_0) = y_0$，$y(x_1) = y_1$。我们的目标是极小化（或极大化）目标泛函：

$$
J[y] = \int_{x_0}^{x_1} F(x, y, y') dx
$$

同时满足约束条件：

$$
K[y] = \int_{x_0}^{x_1} G(x, y, y') dx = C
$$

其中 $F(x, y, y')$、$G(x, y, y')$ 均为三阶连续可微函数，约束条件 $K[y] = C$ 称为“积分约束”，区别于边界条件这类几何约束。

# 2.第二类变分问题求解

## 2.1 拉格朗日乘数法的思想迁移

在多元函数极值问题中，带约束的极值可通过拉格朗日乘数法转化为无约束极值：若求 $f(x, y)$ 在约束 $g(x, y) = 0$ 下的极值，可构造辅助函数 $L(x, y, \lambda) = f(x, y) - \lambda g(x, y)$，通过求解 $L$ 的无约束极值得到原问题的解。

将这一思想迁移到变分问题中，通过引入拉格朗日乘数 $\lambda$（常数，因约束为积分常数），将带积分约束的泛函极值问题转化为无约束的辅助泛函极值问题，再利用欧拉-拉格朗日方程求解。

## 2.2. 辅助泛函

针对单积分约束的第二类变分问题，构造辅助泛函：

$$
\tilde{J}[y] = J[y] - \lambda (K[y] - C) = \int_{x_0}^{x_1} \left[ F(x, y, y') - \lambda G(x, y, y') \right] dx
$$

其中

- $\lambda$ 为待定拉格朗日乘数
- $\tilde{F}(x, y, y') = F(x, y, y') - \lambda G(x, y, y')$ 称为修正被积函数

**小结**：原问题中 $y(x)$ 是 $J[y]$ 在约束 $K[y] = C$ 下的极值函数，当且仅当 $y(x)$ 是辅助泛函 $\tilde{J}[y]$ 的无约束极值函数

## 2.3 辅助泛函的欧拉-拉格朗日方程

辅助泛函 $\tilde{J}[y]$ 为无约束泛函，其极值条件等价于满足欧拉-拉格朗日方程：

$$
\frac{\partial \tilde{F}}{\partial y} - \frac{d}{dx} \left( \frac{\partial \tilde{F}}{\partial y'} \right) = 0
$$

将 $\tilde{F} = F - \lambda G$ 代入，展开得到：

$$
\left( \frac{\partial F}{\partial y} - \lambda \frac{\partial G}{\partial y} \right) - \frac{d}{dx} \left( \frac{\partial F}{\partial y'} - \lambda \frac{\partial G}{\partial y'} \right) = 0
$$

这是一个二阶常微分方程，其通解含两个积分常数和一个待定乘数 $\lambda$，共三个待定参数。

# 3.经典实例：等周问题

## 3.1 问题描述

平面上给定长度为 $L$ 的封闭曲线，求该曲线围成面积最大时的形状（“等周”即周长固定，追求面积最大）。这是古希腊时期就提出的问题，直到变分法发展后才得到严格证明。

## 3.2 构造约束系统

设封闭曲线的参数方程为 $x = x(t)$，$y = y(t)$，$t \in [0, 2\pi]$，满足周期性边界条件 $x(0) = x(2\pi)$，$y(0) = y(2\pi)$（封闭性）。

 目标泛函：平面封闭曲线围成的面积公式为：

$$
J[x, y] = \frac{1}{2} \int_0^{2\pi} (x \frac{dy}{dt} - y \frac{dx}{dt}) dt
$$

约束条件曲线长度为：

$$
K[x, y] = \int_0^{2\pi} \sqrt{ \left(\frac{dx}{dt} \right) ^2 + \left(\frac{dy}{dt} \right)^2} dt = L
$$

## 3.3 求解

**构造辅助泛函**

引入拉格朗日乘数 $\lambda$，辅助泛函为：

$$
\tilde{J}[x, y] = \frac{1}{2} \int_0^{2\pi} \left[ (x \frac{dy}{dt} - y \frac{dx}{dt}) - \lambda \sqrt{ \left(\frac{dx}{dt} \right) ^2 + \left(\frac{dy}{dt} \right)^2} \right] dt
$$

**对 $x(t)$ 和 $y(t)$ 分别列欧拉-拉格朗日方程**

对于二元参数函数的泛函，欧拉-拉格朗日方程形式为：

$$
\begin{align}
\frac{\partial \tilde{F}}{\partial x} - \frac{d}{dt} \left( \frac{\partial \tilde{F}}{\partial \dot{x}} \right) &= 0 \\
\frac{\partial \tilde{F}}{\partial y} - \frac{d}{dt} \left( \frac{\partial \tilde{F}}{\partial \dot{y}} \right) &= 0
\end{align}
$$

计算偏导数并化简后：

$$
\begin{align}
\frac{d^2x}{dt^2} + \frac{1}{\lambda} x = 0, \\
\frac{d^2y}{dt^2} + \frac{1}{\lambda} x = 0
\end{align}
$$

**求解微分方程**

上述方程的通解为：

$$
\begin{align}
x(t) = A \cos\left( \frac{t}{\sqrt{\lambda}} \right) + B \sin\left( \frac{t}{\sqrt{\lambda}} \right) \\
y(t) = C \cos\left( \frac{t}{\sqrt{\lambda}} \right) + D \sin\left( \frac{t}{\sqrt{\lambda}} \right)
\end{align}
$$

$A,B,C,D$ 为积分常数。

**利用边界条件和约束确定参数**

在这一步中，通过边界条件和约束，构造代数方程求解积分常数。

从曲线封闭这个边界条件可以得到约束：

$$
\begin{align}
x(0) = x(2\pi) \\
y(0) = y(2\pi)
\end{align}

$$

封闭曲线光华，所以满足可导性：

$$
\begin{align}
\frac{dx}{dt}(0) = \frac{dx}{dt}(2\pi) \\
\frac{dy}{dt}(0) = \frac{dy}{dt}(2\pi)
\end{align}
$$

加上边长约束条件，可以得出结论：这个封闭曲线是圆。
