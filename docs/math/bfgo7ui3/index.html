1. 基矢量

张量的协变、逆变特性，本质是张量分量随基矢变换的"响应方式"。

1.1 坐标系变换与基矢的关系

设存在两个线性坐标系：

  • 旧坐标系：基矢为 {g1,g2,g3}\{g_{1}, g_{2}, g_{3}\}{g1​,g2​,g3​}（不一定正交，称为协变基矢）
  • 新坐标系：基矢为 {g1′,g2′,g3′}\{g_{1}', g_{2}', g_{3}'\}{g1′​,g2′​,g3′​}

坐标系变换关系为：

gi′=Aijgj(1)g_{i}' = A_{i}^{j} g_{j} \tag{1} gi′​=Aij​gj​(1)

其中 AijA_{i}^{j}Aij​ 是变换矩阵，iii 是新坐标指标，jjj 是旧坐标指标，爱因斯坦求和约定：重复指标自动求和。

1.2 逆变基矢

对任意协变基矢 gig_{i}gi​，存在一组基矢 gjg^{j}gj，满足正交归一条件：

gi⋅gj=δij(2)g_{i} \cdot g^j = \delta_{i}^{j} \tag{2} gi​⋅gj=δij​(2)

这组对偶基矢 {gj}\{g^j\}{gj} 被称为 {gi}\{g_{i}\}{gi​} 的逆变基矢或者对偶基矢。

例：在直角坐标系中，协变基矢 gi=eig_{i} = e_{i}gi​=ei​（单位正交向量），其对偶基矢 gj=ejg^j = e^{j}gj=ej，因此协变 = 逆变；但在斜坐标系中，二者方向不同，必须区分。

2.1 协变度量张量与逆变度量张量

给定流形 M\mathcal{M}M，其协变基矢为 {gi}\{\mathbf{g}_{i}\}{gi​}，逆变基矢 {gj}\{\mathbf{g}^j\}{gj}，协变度量张量 gijg_{ij}gij​，即第一基本形式定义了基矢的内积：

gij=gi⋅gj(3)g_{ij} = \mathbf{g}_{i} \cdot \mathbf{g}_{j} \tag{3} gij​=gi​⋅gj​(3)

它的核心作用是计算向量的长度、角度、面元/体积元，是描述流形"度量结构"的核心张量。

逆变度量张量 gijg^{ij}gij 的定义就是协变张量的逆元，满足：

gikgkj=δji(4)g^{ik} g_{kj} = \delta_{j}^{i} \tag{4} gikgkj​=δji​(4)

2.2 张量类型

二阶张量的类型由两个指标的协变/逆变属性划分，共 3 类。

2.2.1 逆变度量张量的核心作用

gijg^{ij}gij 最核心的功能是实现张量协变分量与逆变分量的转换，规则如下：

一阶张量指标升降

  • 协变分量转逆变分量：vi=gijvjv^i = g^{ij} v_{j}vi=gijvj​
  • 逆变分量转协变分量：vi=gijvjv_{i} = g_{ij} v^jvi​=gij​vj

二阶张量指标升降

以第一皮奥拉-基尔霍夫应力 P\mathbf{P}P 为例，这是个混合指标张量 PijP_{i}^{j}Pij​：

  • 升下标（混合张量 →\to→ 逆变张量）：Pij=gikPkjP^{ij} = g^{ik} P_{k}^{j}Pij=gikPkj​
  • 降上标（混合张量 →\to→ 协变张量）：Pij=gikPjkP_{ij} = g_{ik} P_{j}^{k}Pij​=gik​Pjk​
