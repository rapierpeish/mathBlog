1.双点积的定义

双点积是针对二阶张量定义的一种运算，也称为“双收缩”（Double Contraction）本质是将两个二阶张量的对应分量相乘后，对所有指标进行求和，最终结果为一个标量（区别于张量与向量的点积、张量与张量的单点积）

在三维笛卡尔坐标系下，双点积的严格定义：

设两个二阶张量A\boldsymbol{A}A和B\boldsymbol{B}B，其分量形式分别为AijA_{ij}Aij​和BklB_{kl}Bkl​（指标含义见下文符号解释），则两者的双点积记为A:B\boldsymbol{A}:\boldsymbol{B}A:B，定义为：

A:B=∑i=13∑j=13AijBij\boldsymbol{A}:\boldsymbol{B} = \sum_{i=1}^{3}\sum_{j=1}^{3} A_{ij} B_{ij} A:B=i=1∑3​j=1∑3​Aij​Bij​

双点积的核心是“双重求和”——先对第一个指标求和，再对第二个指标求和，最终将所有对应分量的乘积相加，得到一个没有指标的标量，这也是“收缩”的本质

2.双点积的基本性质

性质1：双点积结果为标量

张量符号：A:B=标量\boldsymbol{A}:\boldsymbol{B} = \text{标量}A:B=标量（无指标、无方向）

指标符号：AijBij=标量A_{ij} B_{ij} = \text{标量}Aij​Bij​=标量（双重求和后，指标全部消去，得到一个数值）

说明：这是双点积与单点积最核心的区别——二阶×二阶张量的单点积A⋅B\boldsymbol{A}\cdot\boldsymbol{B}A⋅B 结果仍为二阶张量，而双点积结果为标量，因此双点积也称为“双收缩”（完全收缩所有指标）。

性质2：交换律（指标对应前提下）

张量符号：A:B=B:A\boldsymbol{A}:\boldsymbol{B} = \boldsymbol{B}:\boldsymbol{A}A:B=B:A 指标符号：AijBij=BijAijA_{ij} B_{ij} = B_{ij} A_{ij}Aij​Bij​=Bij​Aij​

注意：若两个张量的指标不对应（如AijA_{ij}Aij​和BjiB_{ji}Bji​），则交换律不成立。

性质3：与标量的结合律

张量符号：λ(A:B)=(λA):B=A:(λB)\lambda (\boldsymbol{A}:\boldsymbol{B}) = (\lambda \boldsymbol{A}):\boldsymbol{B} = \boldsymbol{A}:(\lambda \boldsymbol{B})λ(A:B)=(λA):B=A:(λB) 指标符号：λAijBij=(λAij)Bij=Aij(λBij)\lambda A_{ij} B_{ij} = (\lambda A_{ij}) B_{ij} = A_{ij} (\lambda B_{ij})λAij​Bij​=(λAij​)Bij​=Aij​(λBij​)

性质4：分配律（对张量加法的分配）

张量符号：A:(B+C)=A:B+A:C\boldsymbol{A}:(\boldsymbol{B} + \boldsymbol{C}) = \boldsymbol{A}:\boldsymbol{B} + \boldsymbol{A}:\boldsymbol{C}A:(B+C)=A:B+A:C 指标符号：Aij(Bij+Cij)=AijBij+AijCijA_{ij} (B_{ij} + C_{ij}) = A_{ij} B_{ij} + A_{ij} C_{ij}Aij​(Bij​+Cij​)=Aij​Bij​+Aij​Cij​

推导：根据加法分配律，Aij(Bij+Cij)=AijBij+AijCijA_{ij}(B_{ij}+C_{ij})=A_{ij}B_{ij}+A_{ij}C_{ij}Aij​(Bij​+Cij​)=Aij​Bij​+Aij​Cij​，双重求和后，等式依然成立；同理可推导(A+B):C=A:C+B:C(\boldsymbol{A} + \boldsymbol{B}):\boldsymbol{C} = \boldsymbol{A}:\boldsymbol{C} + \boldsymbol{B}:\boldsymbol{C}(A+B):C=A:C+B:C。

性质5：指标顺序调整（关键性质）

若两个张量的指标顺序不一致（如A\boldsymbol{A}A的分量为AijA_{ij}Aij​，B\boldsymbol{B}B的分量为BjiB_{ji}Bji​），则双点积结果需根据指标顺序调整，记为：

A:BT=AijBji\boldsymbol{A}:\boldsymbol{B}^\text{T} = A_{ij} B_{ji} A:BT=Aij​Bji​

其中BT\boldsymbol{B}^\text{T}BT表示张量B\boldsymbol{B}B的转置，转置后张量的分量满足(BT)ij=Bji(\boldsymbol{B}^\text{T})_{ij} = B_{ji}(BT)ij​=Bji​。

说明：双点积的核心是“对应指标求和”，即行指标对应行指标、列指标对应列指标，若指标顺序颠倒，需对其中一个张量取转置，才能满足求和条件。

性质6：特殊张量的双点积（单位张量）

设I\boldsymbol{I}I为二阶单位张量，其分量为Iij=δijI_{ij} = \delta_{ij}Iij​=δij​（δij\delta_{ij}δij​为克罗内克符号，满足δij=1\delta_{ij}=1δij​=1当i=ji=ji=j，δij=0\delta_{ij}=0δij​=0当i≠ji\neq ji=j），则：

A:I=IijAij=δijAij=Aii\boldsymbol{A}:\boldsymbol{I} = I_{ij} A_{ij} = \delta_{ij} A_{ij} = A_{ii} A:I=Iij​Aij​=δij​Aij​=Aii​

其中AiiA_{ii}Aii​是张量A\boldsymbol{A}A的“迹”（Trace），记为tr(A)\text{tr}(\boldsymbol{A})tr(A)，即张量主对角线分量之和（A11+A22+A33A_{11}+A_{22}+A_{33}A11​+A22​+A33​）。
推论：I:I=δijδij=3\boldsymbol{I}:\boldsymbol{I} = \delta_{ij} \delta_{ij} = 3I:I=δij​δij​=3（因为i=j=1,2,3i=j=1,2,3i=j=1,2,3时，δij=1\delta_{ij}=1δij​=1，求和后为1+1+1=3）。
