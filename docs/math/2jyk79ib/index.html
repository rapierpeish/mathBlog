黎曼测地线

在黎曼几何中测地线更“本质”的定义给定流形上的初始点 + 初始速度：

点的运动要满足测地方程的约束：

∇γ˙γ˙=0\nabla_{\dot{\gamma}}\dot{\gamma} = 0 ∇γ˙​​γ˙​=0

因为测地线存在且唯一，那就一定能确定一条唯一的测地线 γv(t)\gamma_v(t)γv​(t) 。 对黎曼测地线更形象化的几何解释是：黎曼测地线是给定初始点和速度后，沿曲面运动速度方向不发生旋转所走过的路径，本质上是点在流形空间上的[[曲面的自然标架运动#并行移动|并行移动]]。 对黎曼测地线的代数解释是：给定初始条件，在解空间中寻找满足约束方程的集合。

例子：

  • 球面 S2S^2S2 ：沿大圆方向走一圈回到起点。

指数映射

几何解释

如果 MMM 是个黎曼流形，p∈Mp \in Mp∈M，那么 黎曼几何的指数映射 是

exp⁡p:TpM→M,\exp_{p} : T_{p}M \rightarrow M, expp​:Tp​M→M,

对 v∈TpMv \in T_pMv∈Tp​M，沿着从 ppp 出发，将在流形上得到初速度为 vvv 的黎曼测地线 。指数映射就是切空间上点的运动轨迹投射到流形上。

代数解释

给定李群 SO(3)SO(3)SO(3) 这是个流形， 李代数 so(3)\mathfrak{so}(3)so(3) 是单位元位置处的切空间。在李代数 so(3)\mathfrak{so}(3)so(3) 上指定一个点，就能产生一个速度 vvv 。

给定单位元位置和速度后，就能定义唯一黎曼测地线 R(t)R(t)R(t)。接下来要考察这个问题：从几何方面来看 R(t)R(t)R(t) 是条参数曲线，从代数方面来看，R(t)R(t)R(t) 是否是 SO(3)SO(3)SO(3) 的[[参数子群]]？

证明 R(t)R(t)R(t) 是 SO(3)SO(3)SO(3) 的参数子群

假设约定

SO(3)SO(3)SO(3) 的李代数记作 so(3)\mathfrak{so}(3)so(3)，记 X∈so(3)X \in \mathfrak{so}(3)X∈so(3) 为李代数中的元素，相当于单位元出的速度，假定李代数中运动轨迹速度是匀速的，即 XXX 是个常数。

证明目标：在给定切平面轨迹 XtXtXt ， 在李群中的轨迹 R(t)=exp⁡(Xt)R(t) = \exp(Xt)R(t)=exp(Xt) 是 SO(3)SO(3)SO(3) 的参数子群。

参数子群定义： 设 GGG 是一个李群（光滑流形并带有群结构）。一个 一参数子群 是一个光滑[[群同态]]：

φ:R→G\varphi:\mathbb{R} \rightarrow G φ:R→G

φ\varphiφ 应该满足：

φ(t+s)=φ(t)φ(s),∀t,s∈R\varphi(t+s)=\varphi(t)\varphi(s), \forall t,s \in \mathbb{R} φ(t+s)=φ(t)φ(s),∀t,s∈R

所以我们需要说明：

exp⁡((s+t)X)=exp⁡(sX)exp⁡(tX)(1)\exp((s+t)X) = \exp(sX)\exp(tX) \tag{1} exp((s+t)X)=exp(sX)exp(tX)(1)

(1)式左侧是点在切面沿着固定方向运动路径的积累，所以是加法。 (1)式右侧是切面在流形上运动的旋转积累，旋转是乘法哦，所以等式是成立的。

综上：R(t)=exp⁡(Xt)R(t) = \exp(Xt)R(t)=exp(Xt) 是 SO(3)SO(3)SO(3) 的参数子群。
