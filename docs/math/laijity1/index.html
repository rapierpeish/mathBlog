3.2 大变形薄壳单元计算

约 2958 字大约 10 分钟

壳理论

2026-02-01

1 引言

经典薄板理论（如 Kirchhoff–Love 或 Reissner–Mindlin 壳理论）通常基于小变形假设，即忽略位移梯度的高阶项，从而简化几何关系。然而，在涉及大位移、大转动甚至中等应变的问题中，小变形假设不再适用，必须采用大变形（几何非线性）壳理论。

本讲义旨在：

  • 回顾经典薄壳理论的基本假设；
  • 推广至大变形情形，保留变形前后第一基本形式的完整表达；
  • 引入高阶插值（如 NURBS）提升几何与位移场的描述精度；
  • 采用严格的分析化方式阐述。

2 薄壳理论

2.1 几何设定

参考构型（即变形前的壳体）：

  • 中面由嵌入映射 r:U⊂R2→R3\mathbf{r}:U \subset\mathbb{R}^2\to\mathbb{R}^3r:U⊂R2→R3 描述；
  • 局部坐标 (u1,u2)(u^1,u^2)(u1,u2)；
  • 切向量： ri=∂ir\mathbf{r}_{i}=\partial_{i}\mathbf{r}ri​=∂i​r；
  • 第一基本形式： gij=ri⋅rjg_{ij}=\mathbf{r}_{i}\cdot \mathbf{r}_{j}gij​=ri​⋅rj​。

当前构型（变形后的壳体）：

  • 变形后中面位置： r′=r+u(u1,u2)\mathbf{r}'=\mathbf{r}+\mathbf{u}(u^1,u^2)r′=r+u(u1,u2)，其中 u\mathbf{u}u 为三维位移场；
  • 切向量： ri′=∂i(r+u)=ri+u\mathbf{r}_{i}'=\partial_{i}(\mathbf{r}+\mathbf{u})=\mathbf{r}_{i}+\mathbf{u}ri′​=∂i​(r+u)=ri​+u；
  • 当前度量：gij=ri′⋅rj′g_{ij}=\mathbf{r}_{i}'\cdot \mathbf{r}_{j}'gij​=ri′​⋅rj′​。

2.2 几何方程回顾

在 [[3.1 弹性薄壳的几何方程]] 中推导了 Green-Lagrange 应变张量的一般形式：

Eij(u,t)=12((gij′−gij)+2t(bij−bij′)+t2(ni′⋅nj′−ni⋅nj))E_{ij}(u,t)=\frac{1}{2}\left((g_{ij}'-g_{ij})+2t(b_{ij}-b_{ij}')+t^2(\mathbf{n}_{i}' \cdot \mathbf{n}_{j}'-\mathbf{n}_{i}\cdot \mathbf{n}_{j}) \right) Eij​(u,t)=21​((gij′​−gij​)+2t(bij​−bij′​)+t2(ni′​⋅nj′​−ni​⋅nj​))

在不考虑扭转和薄壳的假设下，Green-Lagrange 应变张量简化为：

Eij(u)=12(gij−gij′)=12(ri⋅uj+rj⋅ui+ui⋅uj)E_{ij}(u) = \frac{1}{2}(g_{ij}-g_{ij}')=\frac{1}{2}(\mathbf{r}_{i}\cdot \mathbf{u}_{j}+\mathbf{r}_{j}\cdot \mathbf{u}_{i}+\mathbf{u}_{i}\cdot \mathbf{u}_{j}) Eij​(u)=21​(gij​−gij′​)=21​(ri​⋅uj​+rj​⋅ui​+ui​⋅uj​)

该近似完全保留了第一基本形式的非线性变化，足以捕捉由中面拉伸诱导的大位移、大转动引起的几何非线性，同时大幅简化计算。

2.3 应力-应变关系

假设材料为各向同性、超弹性，应变能密度 dWdWdW 仅为 EijE_{ij}Eij​ 的函数：

dW=dW(Eij)dW = dW(E_{ij}) dW=dW(Eij​)

使用 [[2.1 二维流形的弹性静力问题]] 中的弹性模量张量公式 Cijkl\mathbf{C}_{ijkl}Cijkl​：

Cijkl=λδijδkl+μ(δikδjl+δilδjk)\mathbf{C}_{ijkl} = \lambda \delta_{ij} \delta_{kl} + \mu (\delta_{ik}\delta_{jl}+\delta_{il}\delta_{jk}) Cijkl​=λδij​δkl​+μ(δik​δjl​+δil​δjk​)

其中 λ,μ\lambda,\muλ,μ 为 Lamé 常数。

3 Saint Venant–Kirchhoff 应变模型

以下是从弹性模量张量 Cijkl\mathbf{C}_{ijkl}Cijkl​ 出发，结合基本假设，对 Saint Venant–Kirchhoff (SVK) 模型的完整推导，得到 SVK 模型下的应变-应力关系，进而计算应变能密度 dWdWdW。

3.1 SVK 模型的核心假设

  1. 材料是线性各向同性弹性的，用 Green–Lagrange 应变张量来描述变形。
  2. 应变能密度是 Green-Lagrange 应变 E\mathbf{E}E 的二次型。

3.2 应力-应变关系

应力-应变关系为：

S=λ(trE)I+2μE\mathbf{S} = \lambda (tr\mathbf{E}) I +2\mu \mathbf{E} S=λ(trE)I+2μE

  • S\mathbf{S}S 是 [[3.1.1 拉格朗日应力度量#第二类皮奥拉-基尔霍夫应力张量|第二 Piola-Kirchhoff 应力张量]]（2PK）；
  • λ,μ\lambda, \muλ,μ 是 Lamé 常数；
  • trEtr \mathbf{E}trE 是应变的迹。

然后通过形变梯度 F\mathbf{F}F 转化为 Cauchy 应力：

σ=1det⁡FFSFT\boldsymbol{\sigma} = \frac{1}{\det \mathbf{F}} \mathbf{F} \mathbf{S} \mathbf{F}^{T} σ=detF1​FSFT

所以 SVK 模型可以理解为：

  • 几何非线性：使用 Green–Lagrange 应变 E\mathbf{E}E；
  • 材料本构线性：2PK 应力 S\mathbf{S}S 与 E\mathbf{E}E 成线性关系；
  • SVK 模型主要用于处理金属板。

3.3 SVK 应变能密度

应变能密度为：

W(E)=12E:C:EW(\mathbf{E}) = \frac{1}{2} \mathbf{E} : \mathbf{C} : \mathbf{E} W(E)=21​E:C:E

分量形式下：

W=12(λ(trgE)2+2μEijEkl)W = \frac{1}{2}(\lambda(tr_{g}E)^2+2\mu E_{ij}E^{kl}) W=21​(λ(trg​E)2+2μEij​Ekl)

使用逆变度量张量连接 EijE^{ij}Eij 和 EklE_{kl}Ekl​：

Eij=gikgjlEklE^{ij} = g^{ik} g^{jl} E_{kl} Eij=gikgjlEkl​

参考构形上的总能量泛函：

II[φ]=∫B0W(E(φ))dAII[\varphi] = \int_{\mathcal{B}_{0}} W(E(\varphi)) dA II[φ]=∫B0​​W(E(φ))dA

dAdAdA 由第一基本形式计算：

dA=det⁡gx1x2dA = \sqrt{ \det g } x^1 x^2 dA=detg​x1x2

  • x1,x2x^1,x^2x1,x2 是局部坐标。

4 刚度矩阵

本节首先给出两种刚度矩阵的定义方式：

  • 从能量视角出发，将刚度矩阵视为总势能泛函在当前构形处的二阶变分。

通过严格推导，最后得到：

  • 刚度矩阵正是能量泛函在配置空间上所诱导的自然度量，而其在有限变形理论中必然分裂为材料刚度与几何刚度两部分；
  • 离散材料刚度计算公式和离散几何刚度计算公式。

4.1 刚度矩阵定义

刚度矩阵的严格定义：

刚度矩阵 KKK 是线弹性系统中，联系广义节点力 FFF 与广义节点位移 qqq 的线性对称正定算子，其数学本质是系统应变能泛函关于节点位移向量的 Hessian 矩阵。

在有限元框架下具体表现为：

K=∫BTCBdAK = \int \mathbf{B}^T\mathbf{C} \mathbf{B} dA K=∫BTCBdA

4.1.1 刚度矩阵作为函数空间上的内积

设中面位移场属于函数空间：

u∈V,V⊂H1(M,R3)\mathbf{u} \in \mathcal{V},\quad \mathcal{V}\subset H^{1}(\mathcal{M},\mathbb{R}^3) u∈V,V⊂H1(M,R3)

在离散情形下，取有限维子空间 Vh\mathcal{V}_hVh​，位移场基函数展开为

uh=∑adiϕi\mathbf{u}_{h} = \sum_{a} d_{i} \phi_{i} uh​=a∑​di​ϕi​

  • did_{i}di​：位移向量分量；
  • ϕi\phi_{i}ϕi​：基函数向量分量。

在 2.1 章中，刚度矩阵 KKK 是直接由函数空间内积给出的，其对应的离散弹性势能写为

Uh=12dT[∫MEIBT(x)B(x)dx]dU_{h}=\frac{1}{2}\mathbf{d}^{T}\left[\int_{\mathcal{M}}EI \mathbf{B}^{T}(x)\mathbf{B}(x) dx \right] \mathbf{d} Uh​=21​dT[∫M​EIBT(x)B(x)dx]d

在此式中：

  • 括号部分就是刚度矩阵 KKK 的离散定义；
  • B(x)\mathbf{B}(x)B(x) 是应变–位移算子在离散基下的矩阵表示。

在有限弹性理论中，系统的力学行为由总势能泛函控制：

Π(u)=∫M0W(E(u))dA−ℓ(u)\Pi(\mathbf{u}) = \int_{M_{0}} W(E(\mathbf{u})) dA - \ell(\mathbf{u}) Π(u)=∫M0​​W(E(u))dA−ℓ(u)

其中：

  • E(u)E(\mathbf{u})E(u) 为 Green–Lagrange 应变张量；
  • W(E)W(E)W(E) 为 SVK 模型下的应变能密度；
  • ℓ(u)\ell(\mathbf{u})ℓ(u) 为外力势。

根据变分原理：

δΠ(u)[δu]=0\delta\Pi(\mathbf{u})[\delta \mathbf{u}] = 0 δΠ(u)[δu]=0

若 u\mathbf{u}u 为平衡解，则总势能在该点的一阶变分为零，其二阶变分自然定义为

δ2Π(u)[δu,v]=δv(δuΠ[u])\delta^2 \Pi (\mathbf{u}) [\delta \mathbf{u},\mathbf{v}]=\delta_{v}(\delta_{u}\Pi[\mathbf{u}]) δ2Π(u)[δu,v]=δv​(δu​Π[u])

因此我们可以反过来得到刚度矩阵的一般定义：

K(δu,v):=δ2Π(u)[δu,v]K(\delta \mathbf{u,\mathbf{v}}) := \delta^2 \Pi (\mathbf{u}) [\delta \mathbf{u},\mathbf{v}] K(δu,v):=δ2Π(u)[δu,v]

接下来将严格推导 δ2Π\delta^2 \Piδ2Π 的结构。

一阶变分 δΠ\delta \PiδΠ：

  • 来源于应变能密度 W(E)W(E)W(E) 的二阶导数；

5.1 前文重要结论

应变能密度的二阶泛函：

δ2Π=∫M0(δE:C:δE+S:δ2E)dA.\delta^2\Pi=\int_{\mathcal{M_{0}}} (\delta E:\mathbb{C}:\delta E+\mathbf{S}:\delta^2E) dA. δ2Π=∫M0​​(δE:C:δE+S:δ2E)dA.

Eij(u)=12(gij−gij′)=12(ri⋅uj+rj⋅ui+ui⋅uj)E_{ij}(u) = \frac{1}{2}(g_{ij}-g_{ij}')=\frac{1}{2}(\mathbf{r}_{i}\cdot \mathbf{u}_{j}+\mathbf{r}_{j}\cdot \mathbf{u}_{i}+\mathbf{u}_{i}\cdot \mathbf{u}_{j}) Eij​(u)=21​(gij​−gij′​)=21​(ri​⋅uj​+rj​⋅ui​+ui​⋅uj​)
