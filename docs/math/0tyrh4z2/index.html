第一基本形式的正定性

第一基本形式是正定的。

使用代数方法证明 只需要将各个分量带入GGG矩阵中就能得到正定性质

  ds2=E(du1)2+2F,du1du2+G(du2)2  ds^2 = E(du^1)^2 + 2F,du^1 du^2 + G(du^2)^2   ds2=E(du1)2+2F,du1du2+G(du2)2

  • F=⟨φu1,φu2⟩F = \langle \varphi_{u^1}, \varphi_{u^2} \rangleF=⟨φu1​,φu2​⟩
  • G=⟨φu2,φu2⟩G = \langle \varphi_{u^2}, \varphi_{u^2} \rangleG=⟨φu2​,φu2​⟩ 写成矩阵形式为：
